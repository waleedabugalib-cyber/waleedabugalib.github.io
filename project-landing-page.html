Nav Anchors Website

A visual web project focused on navigation structure, hierarchy, and layout clarity.

Project Overview

This project explores the use of navigation anchors to improve usability and content flow on a single-page website. The goal was to create a clean, readable layout that allows users to jump between sections smoothly.

My Role

I designed and built the page structure using HTML and CSS, focusing on spacing, hierarchy, and consistent visual rhythm.

Tools and Skills

  • HTML5
  • CSS3
  • Layout and Visual Hierarchy
  • Navigation Design

Key Takeaways

This project helped me strengthen my understanding of how navigation and layout choices impact usability, especially on longer pages with multiple sections.
