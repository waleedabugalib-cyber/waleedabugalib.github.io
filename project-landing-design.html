Landing Page Design

A landing page focused on clarity, hierarchy, and conversion-driven layout.

Project Overview

This project explores how layout, spacing, and typographic hierarchy can guide users through a landing page clearly and efficiently. The goal was to create a clean, focused design that communicates value quickly.

UX and UI Decisions

  • Strong visual hierarchy to guide attention
  • Clear section separation for scannability
  • Centered content for focus and balance
  • Minimal distractions to support conversion

Skills Demonstrated

  • Landing page layout design
  • Typography and spacing
  • HTML structure
  • Content-first visual design
